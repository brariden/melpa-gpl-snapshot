# sprunge.el

Upload pastes to [sprunge.us](http://sprunge.us/).

## usage

### upload the current buffer

```
M-x sprunge-buffer
```

The returned URL will be killed to the ring and printed to the `*Messages*` buffer.

### upload the active region

```
M-x sprunge-region
```

The returned URL will be killed to the ring and printed to the `*Messages*` buffer.
