# Ritz REPL Utils

Library of useful functions for implementing clojure development clients.

## Usage

[API Docs](http://palletops.com/ritz/0.4/repl-utils/api/)

## Install

Add the following to your `:dependencies`:

```clj
[ritz/ritz-repl-utils "0.7.0"]
```
