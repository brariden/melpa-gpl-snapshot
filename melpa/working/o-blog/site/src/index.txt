<col sm="6" md="4">
<thumbnail>
<caption title="Setup">
If you are using el-get it would be very easy to install. All you need to do is to add o-blog to el-get-sources and type:

`M-x el-get-install o-blog`
</caption>
</thumbnail>
</col>

<col sm="6" md="4">
<thumbnail>
<caption title="Publish">
Only one single command:

`M-x o-blog-publish`
</caption>
</thumbnail>
</col>
