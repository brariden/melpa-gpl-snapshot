# shader-mode

[![MELPA](https://melpa.org/packages/shader-mode-badge.svg)](https://melpa.org/#/shader-mode)

Unity shader editing mode for GNU Emacs

## Installation

Install `shader-mode.el` from MELPA with `package.el`

    M-x package-install shader-mode
