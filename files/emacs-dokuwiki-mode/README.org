* Overview
Major mode for DokuWiki document
* Features
** Highlight markup of DokuWiki
** Work with ~outline-magic~
~outline-magic~ enables to handle headings like ~org-mode~. If you load
~outline-magic~, ~dokuwiki-mode~ use it.
