* pixiv-novel-mode.el

  ja: http://inside.pixiv.net/entry/2014/12/19/230810

** Usage
   Manually src =M-x pixiv-novel-mode=.
   Or use file name as =.pxv= or =.pxv.txt=.

** Install
   #+BEGIN_SRC emacs-lisp
;; package.el(MELPA)
(package-install 'pixiv-novel-mode)

;; El-Get
(el-get 'sync 'pixiv-novel-mode)
   #+END_SRC
