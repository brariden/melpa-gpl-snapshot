![Imgur](http://i.imgur.com/i6RLz8Q.png)

## Installation

You can install `helm-zhihu-daily` from [MELPA](http://melpa.org/) with package.el.

<kbd>M-x package-install RET helm-zhihu-daily RET</kbd>

## Usage

<kbd>M-x helm-zhihu-daily RET</kbd>
